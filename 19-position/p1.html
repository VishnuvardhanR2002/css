CSS POSITION..!

positioning element apporopirately on the web page is a necessity for a good layout design

There are 4 values

  1. RELATIVE - Current position and reference
  2. ABSOLUTE - body as a reference and scroll along with content
  3. FIXED - Body as a ref, will not scroll along with content
  4. STICKY - depend on the scrolling position

position 4 properties

  1. TOP
  2. RIGHT
  3. BOTTOM
  4. LEFT
