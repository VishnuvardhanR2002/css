CSS pseudo-elemeny

The css pseudo-elemenet allows you to style the elements or part of the elements without adding any ids or classes to them

CSS3 introduces new double-colon(::) syntax for pseudo elements to distinguish b/w them and pseudo-classes

selector::pseudo-element {property : value;}

::first-line ::first::letter
