CSS PSEUDO-CLASSES

The css pseudo-classes allow yuo to style the dynamic states of an element such as hover, active and focus state as well as element that are existing in the document tree but cant be targeted vis the use of other selectors without adding any ids or classes to them for ex: targeting the first or the last element

A pseudo-class starts with a colon( :). its syntax can be given with:

selector:pseudo-class { property: value;}
